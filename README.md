## Key Projects

### [Event Management System](https://github.com/6-Brain-Cells/event-management-cloud) · Cloud & DevOps Course
- Architected a production-grade **4-service FastAPI** platform behind an **Nginx Gateway** enforcing rate limiting, correlation-ID tracking, and **JWT + RBAC** security
- Engineered resilient async messaging via **RabbitMQ Topic Exchanges** with custom **Dead Letter Queues**, **circuit breakers** for HTTP protection, and version-based **optimistic concurrency**
- Automated cloud-native deployment utilizing **Kubernetes Helm charts** and **ArgoCD GitOps** alongside a unified **Prometheus, Grafana, and Loki** observability stack

### [TailorCV](https://github.com/AbanoubPhelopos/TailorCV) · AI-Powered CV Platform
- Architected a **Modular Monolith** in **.NET 10** using **Vertical Slice Architecture** and **CQRS**, with per-module PostgreSQL schemas enforcing strict domain boundaries
- Engineered hybrid inter-module communication: **Wolverine + RabbitMQ** for resilient async workflows (sagas, outbox pattern) and **gRPC** for synchronous retrieval calls
- Automated a data pipeline combining **OpenAI API** for resume parsing and semantic job matching, **Playwright** for headless job scraping, and **Puppeteer** for HTML-to-PDF rendering

### [E-Commerce Microservices](https://github.com/AbanoubPhelopos/E-Commerce)
- Architected a distributed platform of **6 ASP.NET Core 10** microservices (Basket, Catalog, Discount, Ordering, UserManagement, Payment) behind an **Nginx** reverse proxy and **Ocelot** API gateway with **Clean Architecture** and **CQRS** via **MediatR**
- Engineered polyglot persistence (**MongoDB**, **Redis**, **PostgreSQL + Dapper**, **SQL Server + EF Core**), async saga-based checkout flow via **MassTransit + RabbitMQ** (Basket → Ordering → Payment → **Stripe**), and synchronous discount evaluation over **gRPC**
- Secured all services with **Duende IdentityServer** (OAuth2 / OpenID Connect) and a shared **Redis-backed idempotency middleware** preventing duplicate processing across all state-changing operations

### [Wsnly](https://github.com/AbanoubPhelopos/Wsnly-Backend) · Graduation Project — AI Transit Platform
- Architected a multi-service AI transit platform using **Django**, **Python**, and **C++** microservices orchestrated over **gRPC** for high-performance inter-service communication
- Engineered an **NLP** pipeline to extract structured location data from natural language queries, paired with a custom **C++ A\* routing engine** computing optimal paths over in-memory GTFS data in **< 15 ms**

### [FOS — FCIS Operating System](https://github.com/fcisProjects/FOS)
- Built a modular OS kernel featuring dynamic memory allocation (First-Fit / Best-Fit), virtual memory paging, and an optimised **Nth Chance Clock** page-fault handling algorithm
- Implemented process management with a **Priority Round-Robin** scheduler and synchronisation primitives (SpinLocks, Sleep Locks, Semaphores) for safe inter-process communication
